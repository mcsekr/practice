# springboot-encryption-app

## Requirements
- [JDK 1.8](http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html)
- [Maven 3](https://maven.apache.org)

## Clean the application 

```shell
mvn clean
```

## package application 

```shell
mvn package
```

## Running the application 

```shell
mvn spring-boot:run
```
